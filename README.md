🦊 Crash Bandicoot AI Quiz

Prove your gaming knowledge… or get emotionally destroyed trying.

🧠 What Is This?

Crash Bandicoot AI Quiz is a Tkinter-based game where:

🤖 AI generates questions about Crash Bandicoot

🎮 You answer with Yes or No

🧾 Your score determines your fate

It’s fast.
It’s unfair.
It knows if you’re a fake fan.

🎮 How It Works

The game asks you 5 AI-generated questions

Each question is:

About Crash Bandicoot

A Yes/No question

Designed for “experienced gamers” (uh oh)

You answer using the buttons:

✅ Yes

❌ No

No hints. No help. Just vibes.

🤖 AI Question Engine

Powered by:

gpt-3.5-turbo

A suspiciously strict system prompt

Each question comes in this format:

Question: <question> | Answer: <yes/no>

Only one is correct.
And the AI will not explain itself.

🧮 Scoring System

After 5 questions, the game reveals your true identity:

0 points → 🌪️ "You should get spinned"

1 point → 😬 "WOAH this is bad"

2 points → 😵 "Not well not well not well"

3 points → 💪 "You are filled with DETERMINATION"

4 points → 🍎 "You won a wumpa fruit!!!!"

5 points → 🏆 "PLATINUM"

No pressure.

🎯 Features

🤖 Fully AI-generated questions

😈 Brutal honesty in scoring

🧠 Tests real knowledge (or lucky guessing)
